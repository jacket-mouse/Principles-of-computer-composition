# CPU的寄存器

## 用户可见寄存器

- 通用寄存器
	存放操作数或操作数的地址，可作某种寻址方式所需的专用寄存器。
- 数据寄存器
	存放操作数（满足各种数据类型）两个寄存器并接存放双倍字长数据
- 地址寄存器
	存放地址，其位数应满足最大的地址范围，用于特殊的寻址方式 段基址 栈指针
- 条件码寄存器
	存放条件码，可作为程序分支的依据 如正、负、零、溢出、进位等
## 控制和状态寄存器

- 控制寄存器
	- PC->MAR->M->MDR->IR
	- 控制CPU操作
	- MAR、MDR、IR用户不可见
	- PC用户可见
- 状态寄存器
	- PSW寄存器 存放程序状态字
